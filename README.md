# Weather Dashboard

A responsive weather dashboard built with HTML, CSS, and JavaScript.

## Features

- Search weather by city or place name
- Displays current temperature, condition, humidity, and wind speed
- Handles empty input, invalid locations, and network failures
- Works without a private API key by using Open-Meteo APIs

## Run Locally

1. Open the project folder.
2. Start a simple local server from that folder.
3. Visit the local server URL in your browser.

Example using Python:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Project Structure

- `index.html` - page structure
- `style.css` - responsive styling
- `index.js` - search and weather logic
- `images/` - weather icons
- `certificate/` - internship certificate PDF
